人事面试 设计模式

  • work

more便签不能用？？

1.自我介绍

面试官们好，我叫王正进，是江苏师范大学计算机科学与技术专业的。我挺喜欢计算机，也会在网络上自学一些这方面的技术。平时喜欢听听音乐，和朋友们交流生活心得。我近两年的计划是想成为一个独立的后台工程师。

2.个性上的优点

坚持，自从下定学习英语的决心，我就坚持每天打卡。换位思考，会站在别人的角度思考。

3.个性上的缺点

每当看见一些有趣的技术，总是想花一番功夫学习。但是由于心急，总是让自己手忙脚乱。现在我给了这些任务划分处理有限级别，优先处理高等级的任务。

4.为什么从上一家离职

之前公司的氛围很棒，团队小伙伴都很年轻，大家相处的很融洽，我在那接触和学习到很多东西。但是，公司对于应届生没有留职名额，所以我不得不离开。我从这家公司学到了一些JavaWeb的开发技能，我希望可以应用在新的工作中。
